📝 Problem:	Problem Statement		

Given an `annual_payments` table, answer the following via SQL and output as a table:
1. How many total transactions are in this table?
2. How many different users made transactions?
3. How many transactions listed as 'paid' have an amount >= 100?
4. Which product made the highest revenue? (use only 'paid' transactions)

Tables: annual_payments(id INT, amount FLOAT, created_at DATETIME, status VARCHAR, user_id INT, amount_refunded FLOAT, product VARCHAR, last_updated DATETIME)
Output: question_id INT, answer TEXT			
			
📌 Notes / Hints:	Key concepts, mistakes, follow-ups	

Mistake: using Group by product having sum(amount) = Max
Max is a funciton so need to add a subquery in order for it to give us the answer we need.
Using order by addresses our purpose without losing optimized solution. 
Also you cannot do Max(Sum()) as you cannot aggregate an aggregate function
