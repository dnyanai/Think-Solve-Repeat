📝 Problem:	Problem Statement		

Write a query to determine the top 5 actions performed during November 2020 on Apple platforms (iPhone, iPad).

The events table tracks every time a user performs an action on a platform.
Output the action and its rank in ascending order. Equal actions get the same rank.

Tables: events(user_id INT, created_at DATETIME, action VARCHAR, platform VARCHAR)
Output: action VARCHAR, ranks INT
