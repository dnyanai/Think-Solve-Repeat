📝 Problem:	Problem Statement		

Given the transactions table, write a query to get the max quantity purchased for each distinct product_id, every year.

Output: year INT, product_id INT, max_quantity INT — sorted by year and product_id ascending.

Tables: transactions(id INT, user_id INT, created_at DATETIME, product_id INT, quantity INT)
